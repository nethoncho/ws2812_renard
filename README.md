ws2812_renard
=============

A WS2812 / Neo Pixel controller using the Renard Protocol.

Tested With
---------
* Open Lighting Architecture: https://github.com/OpenLightingProject/ola
* Vixen Lights: http://www.vixenlights.com/

Default Settings
---------
* LED Count: 60
* Baud Rate: 115200
* Output Pin: 11
